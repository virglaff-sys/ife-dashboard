# IFE Dashboard - Портрет продукта

Статический сайт с портретом продукта бортовых развлекательных систем.

## Назначение
Стратегический отчёт с анализом базовых функций, комплектования борта и системы отчётности.

## Технологии
- HTML5
- CSS3
- JavaScript (Chart.js)
- Font Awesome

## Доступ
Сайт доступен по адресу: https://virglaff-sys.github.io/ife-dashboard/
